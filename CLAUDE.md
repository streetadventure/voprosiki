# Вопросики — Карточки для душевных разговоров

## Описание проекта

Веб-приложение с карточками вопросов для тёплых разговоров с семьёй, друзьями и близкими. Простой одностраничный сайт без бэкенда.

**Репозиторий:** https://github.com/streetadventure/voprosiki
**Деплой:** Vercel (автоматический из main ветки)

## Технологии

- Чистый HTML/CSS/JS (один файл `index.html`)
- Google Fonts (Nunito — кириллический шрифт)
- Web Audio API для звука перелистывания
- CSS анимации для градиентов и свайпов

## Структура

```
/
├── index.html      # Всё приложение
├── vercel.json     # Конфиг для Vercel
└── CLAUDE.md       # Этот файл
```

## Функционал

### Вопросы
- 165 вопросов в 11 категориях
- Категории: Благодарность, Мечты, Ценности, Рост, Детство, Отношения, Творчество, Радость, Размышления, Приключения, Вдохновение

### Режимы
- **Все** — все вопросы по порядку
- **Рандом** — все вопросы перемешаны, фон меняется под категорию текущего вопроса
- **По категориям** — только вопросы выбранной категории

### Навигация
- Свайп влево/вправо (тач и мышь)
- Клавиатура: стрелки ← →, пробел

### Дизайн
- Карточка на всю высоту экрана (100dvh)
- Анимированные градиентные фоны (переливаются)
- 11 цветовых схем для категорий
- Адаптивный дизайн (мобильный/десктоп)
- Звук "свуш" при перелистывании

## Ключевые CSS-решения

- `height: 100dvh` — правильная высота на мобильных (учитывает адресную строку)
- `flex: 1` на card-container — карточка занимает всё свободное место
- `background-size: 400% 400%` + анимация — эффект переливающегося градиента
- `overflow-x: auto` на категориях — горизонтальный скролл в одну строку
